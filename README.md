# Водить.РФ — Платформа бронирования морского транспорта

Этот проект разработан с использованием **Python (FastAPI)**, **SQLite** и **Tailwind CSS**.

## Быстрый старт (Для разработчиков)

Проект изначально подготовлен к легкому развертыванию (Plug and Play). Автоматически создаются таблицы баз данных и административный доступ.

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python (версии 3.9+). Откройте терминал в папке проекта и создайте виртуальное окружение:

**Для Windows:**
```bash
python -m venv .venv
.\.venv\Scripts\activate
```

**Для macOS / Linux:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Установка зависимостей
Установите все необходимые библиотеки (FastAPI, Pydantic, Uvicorn и др.):
```bash
pip install -r requirements.txt
```

### 3. Запуск сервера
Запустите основной файл:
```bash
python main.py
```

### 4. Автоматическая инициализация Базы Данных SQLite
Вам **не нужно** вручную создавать базу данных. При первом запуске `main.py` система автоматически:
1. Создаст файл базы `database/drive.db`.
2. Выполнит миграции из `database/schema.sql`.
3. Добавит 3 доступных вида транспорта из папки `assets/`.
4. Создаст учетную запись администратора.
